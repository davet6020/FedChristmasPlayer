Hello and welcome to this harmless music player. Here are some things to know about it.

To start it:
    double-click on runme.cmd

To stop it:
    Click the Close button

The songs it plays:
    There is a Songs directory. The .mp3 files in this directory will be played in 
    alphabetic order. If you want to add more .mp3 songs to this directory, they
    will play in alphabetic order once you close the app and restart it.
